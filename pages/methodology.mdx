## Data updating

When dealing with current year data, the figures shown are an annual projection using the already available months.

```
Example as follows...
Current year, indicator: 2021, Home sales
Available months: January (1000 home sales), February (2000 home sales)
Projected figure: Available months average * total months -> ((1000+2000/2)*12
Figure shown for 2021: 18.000 home sales
```

As a rule of thumb, the data is updated quarterly. In any case, the last revised date can be consulted in the header of each indicator post.
